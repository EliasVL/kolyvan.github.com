--- layout: kybook_post title: Version 1.6.7 ---

KyBook v1.6.7 is now available on the App Store.

What's New in this Version

  • Completely new rendering engine for FB2 format (ios 7 only)
  • Full support of RTF format (ios 7 only)
  • Improved zooming mode for DJVU/PDF
  • Auto margin cropping by double tap for DJVU/PDF
  • Manual margin cropping by double touch for DJVU/PDF works better
  • Optimizing memory usage in the background
  • Fixed an issue with the loss of page slider for DJVU/PDF
  • Fixed "Wrong Content-Type" error while loading DJVU file
  • Fixed a crash when transferring large files into KyBook from another applications
