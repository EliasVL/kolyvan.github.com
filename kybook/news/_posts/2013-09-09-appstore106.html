--- layout: kybook_post title: Version 1.0.6 ---

KyBook v1.0.6 is now available on the App Store.

What's New in this Version

  • Theme editor allows to create new color themes for ebook, adjust text color, background color and set an image as background pattern.
  • Settings completely redesigned
  • Added a setting of indentation
  • Added the feature to display the system time and battery level in a page header
  • Improved handling of footnotes in e-books
  • Fixed an failure uploading files into Yandex.Disc
