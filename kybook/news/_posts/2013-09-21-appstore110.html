--- layout: kybook_post title: Version 1.1.0 ---

KyBook v1.1.0 is now available on the App Store.

What's New in this Version

  • Experimental support of RTF format
  • Fixed the issue 'Unable to open DJVU file after moving'
  • Integration with social networking services (Facebook, Twitter, Vkontakte)
  • Supports of VK Docs as Cloud Storage
  • Ability to open ebooks in other apps
  • Redesign editing of book's metadata (title, subject, authors and cover)
  • Compatibility improvements with future iOS versions
  • Updated russian hyphenation dictionary
  • Allows user to add custom fonts
  • Allows user to specify the SMB path manually
  • Fixed an issue with the search, if the there are hyphenation in a text
  • Fixed an issue with the TTS, if the there are hyphenation in a text
  • Fixed an issue with upload into Yandex.Disc if the folder name contains non-latin characters
  • Various minor changes and improvements
